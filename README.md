🌟 UKUSA Luxury Stays & Reservations

UKUSA is a luxury hotel booking web application built using HTML, CSS, and JavaScript. It provides an elegant interface for users to explore premium stays, event venues, dining options, and make reservations with invoice generation.

🚀 Features
🏨 View Luxury Stays & Suites
🎉 Explore Event Venues & Banquets
🍽 Filter Dining Menu (Veg / Non-Veg)
📅 Book reservations via form
🧾 Auto-generate Reservation Invoice
📱 Fully Responsive Design
⚡ Smooth navigation & UI effects
🛠 Technologies Used
HTML5 – Structure
CSS3 – Styling & layout
JavaScript (Vanilla) – Dynamic functionality
